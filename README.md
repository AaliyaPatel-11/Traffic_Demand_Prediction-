# 🚦 Traffic Demand Prediction Dashboard

An AI-powered web application that predicts hourly traffic volume using Machine Learning. The dashboard is built with **Streamlit** and uses a trained **LightGBM Regressor** to estimate traffic demand based on weather and time-related features.

---

## 📌 Project Overview

Traffic congestion is a major challenge in urban areas. This project predicts traffic volume using historical traffic and weather data, helping users estimate congestion levels and make informed travel decisions.

The application provides an interactive dashboard where users can input weather conditions and time information to receive real-time traffic predictions.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/AaliyaPatel-11/Traffic_Demand_Prediction-.git
```

Move into the project folder

```bash
cd Traffic_Demand_Prediction-
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
